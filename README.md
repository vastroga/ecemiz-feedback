# Ecemiz Sürücü Kursu - Müşteri Geri Bildirim Sistemi

Bu proje, Ecemiz Sürücü Kursu müşterilerinin deneyimlerini değerlendirmeleri ve sorunlarını doğrudan iletebilmeleri için tasarlanmış modern bir geri bildirim arayüzüdür.

## Özellikler

*   **Modern Arayüz:** Cam efektli (Glassmorphism), mobil uyumlu ve şık tasarım.
*   **Hızlı Değerlendirme:** Memnun müşteriler için tek tıkla Google Haritalar yorum sayfasına yönlendirme.
*   **Sorun Bildirimi:** Sorun yaşayan müşteriler için WhatsApp üzerinden doğrudan iletişim kurma imkanı.
*   **Sosyal Medya:** Instagram profiline hızlı erişim.

## Proje Yapısı

```
ecemiz-feedback/
├── css/
│   └── style.css       # Tüm stil dosyaları
├── js/
│   └── script.js       # İnteraktif özellikler ve WhatsApp entegrasyonu
├── assets/
│   └── images/
│       └── logo.png    # Kurum logosu
├── index.html          # Ana sayfa
└── README.md           # Proje dokümantasyonu
```

## Kurulum ve Kullanım

Bu proje statik bir web sitesidir. Kullanmak için:

1.  Proje dosyalarını bilgisayarınıza indirin.
2.  `index.html` dosyasını tarayıcınızda açın.

Web sitenizde yayınlamak için dosyaları sunucunuza yüklemeniz yeterlidir.

## Geliştirme

Stil değişiklikleri için `css/style.css` dosyasını, mantıksal değişiklikler (telefon numarası güncelleme vb.) için `js/script.js` dosyasını düzenleyebilirsiniz.
